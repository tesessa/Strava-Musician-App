# @strava-musician-app/backend

Backend application built with Next.js and TypeScript.

## Development

Run the dev server from the repo root:

```bash
npm start
```

Or run only the backend:

```bash
npm run dev -w apps/backend
```

The app runs on http://localhost:3001 by default.

## API Endpoints

- `GET /api/health` - Health check endpoint

## Building

```bash
npm run build -w apps/backend
```
